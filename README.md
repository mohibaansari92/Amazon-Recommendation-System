# Amazon Recommendation System  

This project is a machine learning–based "Amazon Product Recommendation System".  
It analyzes Amazon customer reviews, extracts meaningful features, and predicts star ratings using "XGBoost" with TF-IDF and sentiment analysis.  

---

# What This Project Does  
- Cleans and processes Amazon reviews data  
- Extracts extra features like helpfulness ratio, review length, sentiment score, user/product average scores, and time-based features  
- Uses TF-IDF on review text to capture important words  
- Trains an **XGBoost model** to predict star ratings (1–5 stars)  
- Provides visualizations and saves predictions for further analysis  

---

# Results  
- Accuracy: 94%  
- Weighted F1 Score: 0.94
